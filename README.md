# Emberglow Universe — Official Repository

The **Emberglow Universe** is a growing fantasy franchise built around cinematic storytelling, mythic worldbuilding, and a cohesive cross‑media brand. This repository contains the source files, manuscripts, website code, and creative assets that power the Emberglow experience.

## ✨ Overview

Emberglow is a fantasy saga about **ancient fire‑born magic**, **interwoven destinies**, and a world shaped by the mysterious force known as the Emberglow.

This project spans:

- A multi‑book narrative series  
- A definitive lore and worldbuilding guidebook  
- Character compendiums  
- Visual galleries and illustration prompts  
- A full franchise website  
- Tools and workflows for expanding the universe

This repository serves as the **central hub** for development, collaboration, and versioning.

## 🔥 Key Features

- **Cinematic Worldbuilding** — High‑detail lore, mythologies, histories, and character arcs.  
- **Franchise‑Level Branding** — Orange‑and‑black visual identity, consistent typography, unified design language.  
- **Production‑Ready Website** — Fully responsive HTML/CSS site built for Netlify deployment.  
- **Illustration Prompt Library** — Pixar‑meets‑Ghibli style prompts for generating consistent artwork.  
- **Print‑Ready Manuscripts** — Structured, clean, and formatted for publication.

## 🚀 Getting Started

### Clone the Repository
git clone https://github.com/<your-username>/emberglow.git
cd emberglow

### Run the Website Locally
Open `website/index.html` in your browser or serve it with any static server.

### Editing Manuscripts
All manuscripts are stored in Markdown or Word‑compatible formats for easy editing and version control.
